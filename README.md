# ETH 合约交易中枢

## 一键部署

1. 把整个文件夹放到 `C:\Users\你的用户名\`
2. 安装依赖：
   - Node.js: https://nodejs.org (LTS)
   - Python: https://python.org (3.12+)
   - Git: https://git-scm.com
3. 双击运行 `setup.bat`
4. 配置 Clash Verge + 机场订阅
5. 到 OKX/Tebbit 创建只读 API Key
6. 把 API Key 告诉 Claude

## 包含
- Alpha Skills 113 量化技能
- 仓位计算器 / 场景分析器 / 技术分析器
- ETH 实时数据管道
- 恐惧贪婪指数脚本

## 使用
跟 Claude 对话即可，说"查行情"或"帮我分析 ETH"就能调用全套框架。
